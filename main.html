Visualizing United States Census & Per Capita Data

Plot-1 (States are clickable)

Mark : Map (2-D) Area, Channel : Color

Here we can see per capita income in each state and total income of each state in two separate plots. Clicking on button(Per capita) gives plot of united states map with all states with color hue changing from light red to dark red based on income of state. Clicking on button(Raw) gives plot of united states map with total income all states with color hue changing from light blue to dark blue based on income of state.

I took the stats of year 2017. For total income I muliplied population (year 2017) of that state with per capita income.

Using different Buttons on above graph1, changes the plot. If we select per capita button, it gives map of all states with per capita income. If we select raw button, it gives map of all states with total income of each state.

Plot-2 (Plot-2 changes on selecting a state in plot-1)

Mark and Channel : line and Position

Here we can see the population growth each state in last 7 years. This plot is for each state. It can be updated by selecting map for each state.

For example, if we select Texas State after clicking raw button, The plot updates to Texas State and displays number of population in thousands starting from year 2011 and going to year 2017.

Linkage Between Plot 1 and 2 :

Selecting a state on first plot with any map (Raw or capita) , gives population statistics of each state (wrt clicking state) from year 2011 to year 2017.
Example : Click on per capita and then state New York. Plot2 Automatically changes for New York State.
